# cronn commons-lang #

Add the following Maven dependency to your project:

```xml
<dependency>
    <groupId>de.cronn</groupId>
    <artifactId>commons-lang</artifactId>
    <version>1.0</version>
</dependency>
```

## Requirements ##

- Java 17+
